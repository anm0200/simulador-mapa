Resumen de la visualización

Esta vista principal representa la simulación temporal de trayectorias aéreas a partir del conjunto de datos proporcionado.

Qué se representa

Se muestran rutas completas, tramos recorridos y posición actual de cada vuelo según el instante simulado.

Temporalidad

La simulación sigue una línea de tiempo global para representar la evolución conjunta de todos los vuelos.

Finalidad

Esta visualización sirve como base para aplicar métodos, analizar trayectorias y estudiar patrones espaciales y temporales.
